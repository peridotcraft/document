# PERIDOT CRAFT DOCUMENT SITE

## USAGE

compass: `$ make compass`

server: `$ hexo server`

build: `$ hexo generate`

deploy: `$ hexo deploy`

## ページの追加

記事は全てsourceの中に追加していきます。



## 画像の取り扱い

themes/casper/images/配下においた画像はURL`/images/`から参照可能です。

例:

themes/casper/images/hoge.png -> `<img src="/images/hoge.png">`
